## Requisitos
- Node.js 18+
- npm 9+

## 1) Instalar dependencias
```bash
npm ci
```
## 2) Iniciar fronted + api (Angular + json-server)
```bash
npm run dev
```
## 3) Iniciar solo API mock (json-server)
```bash
npm run api
# La API queda en http://localhost:3001
```
Datos iniciales en `db.json` (vets, owners, pets, appointments, records, users).

## 4) Iniciar solo frontend (Angular)
```bash
npm start
# Navegar a http://localhost:4200
```
